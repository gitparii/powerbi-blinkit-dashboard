# Blinkit Sales Dashboard 📊

## 📌 Overview

This project is a Power BI dashboard built using Blinkit sales dataset to analyze business performance and generate insights.

## 📂 Project Structure

* `dashboard/` → Power BI file (.pbix)
* `data/` → Dataset used for analysis
* `images/` → Dashboard screenshots

## ⚙️ Tools Used

* Power BI
* DAX (Data Analysis Expressions)
* Excel / CSV Dataset

## 📊 Features

* KPI Metrics (Total Sales, Profit, Orders)
* Category-wise Sales Analysis
* Region-wise Performance
* Time-based Trends

## 📸 Dashboard Preview

![Dashboard](images/dashboard.png)

## 📈 Key Insights

* Highest sales observed in top-performing regions
* Certain product categories contribute majority of revenue
* Seasonal trends visible in sales data

## ▶️ How to Use

1. Download the `.pbix` file from `dashboard/`
2. Open in Power BI Desktop
3. Load dataset if required
4. Explore dashboard visuals
